Welcome to CloudML documentation!
=================================

CloudML is a Python package for machine learning on top of `scikit-learn <http://www.scikit-learn.org>`_ and `NumPy <http://www.numpy.org/>`_ / `SciPy <http://www.scipy.org/>`_ libraries.
CloudML provides you with a set of tools to build and train various classifiers in a cloud.
